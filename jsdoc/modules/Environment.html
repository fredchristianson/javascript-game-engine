Class: Environment

Environment()

Environment name/value property provider. Multiple sources of property values are searched in priority order.

  1. Set with Environment.set()

  2. Url parameters: This url http://localhost:4263?env=dev&performance=hi sets properties for "env", and "performance"

  3. /games/game/CURRENT-GAME/env.json

  4. JSON files loaded with loadUrls(url1,url2,...). Order of urls is the priority.

(async) loadUrls(…urls)

loads properties from one or more urls. if multiple urls have the same property, the first one has priority.

If a load fails, the urls is ignored. It's not an error to try to load a url that doesn't exist.
